# Go Book Library Web App

## Description
This project is a simple web application written in Go that allows users to view a list of books, see details about each book, and access a contact page.

## Features
- Home page listing all books
- Book detail page
- Contact page

## How to Run

### Installation
1. Clone the repository:
   ```sh
   git clone https://github.com/Ashen-ui/exam-git-avance-luka.git
   cd exam-git-avance-luka
   ```
### Usage
1. Start the server:
   ```sh
   go run main.go
   ```
2. Open your browser and go to [http://localhost:8080](http://localhost:8080)

### Routes
- `/` : Home page (list of books)
- `/book?id=1` : Book detail page (replace `1` with the book ID)
- `/contact` : Contact page
